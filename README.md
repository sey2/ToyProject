### 문제 접근 1 <br>
#### ``G W i " <End> <Ctrl + @> ZZ ``  <br> 
위의 커멘드대로 **8점**이 나왔습니다.  <br><br> 

**커멘드 설명** 
|옵션|설명|
|:---:|:---:|
|G|문서 맨 아래로 이동 합니다. |
|W|공백 기준으로 다음(단어의 시작)으로 이동 합니다.|
|i|커서 위치에 문자를 삽입 합니다.|
|End|해당 라인의 맨 끝으로 이동 합니다. (편집 모드에서도 가능)|
|Ctrl+@|이전에 삽입된 텍스트를 삽입하고 편집 모드를 중지합니다.|
|L|현재 화면의 마지막 라인으로 이동|
